## 📂 Arquitetura do Portfólio e Reprodutibilidade

> 🔒 **Nota de Governança de Dados (Compliance):** Para garantir o sigilo e a segurança da informação de dados corporativos sensíveis, os repositórios abaixo utilizam **scripts de geração sintética automatizada**. Ao executar os pipelines, os arquivos físicos originais em Microsoft Excel (`.xlsx`) são gerados de forma mascarada, garantindo a execução integral dos códigos SQL e Python sem expor dados proprietários.

| 🛠️ **Habilidades** | Pipelines de ETL, SQL Avançado (CTEs e Window Functions), Python (Pandas), Power BI e Excel Avançado.
| 📂 **Projetos** | Repositórios com pipelines funcionais completos que geram seus próprios arquivos de teste e bancos de dados locais.
